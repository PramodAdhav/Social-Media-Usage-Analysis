Social Media Analysis Dashboard

This dashboard provides insights into social media usage patterns based on collected data from various sources. We utilized Google Sheets for data collection and management and employed MS SQL for backend database management. The website is developed using frontend technologies such as Bootstrap, HTML, and CSS. The backend is powered by Flask, a Python-based web framework, and MS SQL for database management. Visualization of data is facilitated using Matplotlib, Seaborn, and pandas libraries.
